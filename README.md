# Python Card Game Collection

A collection of classic card games built with Python 3, focusing on Object-Oriented Programming (OOP) and clean logic.

## 🚀 Projects Included

### 1. Automated War Simulation
An algorithmic representation of the card game 'War'.
- **Challenge:** Managing deep recursion/loops during "Tie" events.
- **Skills:** Collection management, list manipulation, and simulation logic.

### 2. Interactive Blackjack
A command-line interface (CLI) game that allows for player interaction.
- **Challenge:** Implementing an Ace-value adjustment algorithm and a stateful betting system.
- **Skills:** User input validation, OOP encapsulation, and conditional state logic.

## 🛠️ Technical Skills Demonstrated
* **Encapsulation:** Using private methods (e.g., `_adjust_for_aces`) to hide internal logic.
* **Error Handling:** Robust `try/except` blocks for user betting and game flow.
* **OOP Architecture:** Clear separation of concerns between `Card`, `Deck`, `Hand`, and `Player` classes.c
